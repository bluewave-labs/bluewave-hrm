## Features 

Headcount is an open source human resources management (HRM) application for your organization. It has the following features: 

- Team management
- Employee management
- Time off & vacation management
- Onboarding
- Offboarding
- Reporting
- Employee logs
- Surveys

## Tech stack

* [ReactJs](https://react.dev/)
* [MUI (React framework)](https://mui.com/)
* [Node.js](https://nodejs.org/en)
* [PostgreSQL](https://www.postgresql.org/)

This application does not rely on any 3rd party proprietary software to run.
